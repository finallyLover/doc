### 安装

```
#dockers安装

docker run -dit \
  --name QL \
  --hostname QL \
  --restart always \
  -p 5700:5700 \
  -v $PWD/QL/config:/home/data/ql/config \
  -v $PWD/QL/log:/home/data/ql/log \
  -v $PWD/QL/db:/home/data/ql/db \
  -v $PWD/QL/scripts:/home/data/ql/scripts \
  -v $PWD/QL/jbot:/home/data/ql/jbot \
  whyour/qinglong:latest
```
